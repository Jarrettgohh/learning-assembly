## Common GPRs (General Purpose Registers)

| 64-bit register | Lowest 32-bits | Lowest 16-bits | Lowest 8-bits |
| --------------- | -------------- | -------------- | ------------- |
| rax             | eax            | ax             | al            |
| rbx             | ebx            | bx             | al            |
| rcx             | ecx            | cx             | cl            |
| rdx             | edx            | dx             | dl            |
| rsi             | esi            | si             | sil           |
| rdi             | edi            | di             | dil           |
| rbp             | ebp            | bp             | bpl           |
| rsp             | esp            | sp             | spl           |
